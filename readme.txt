== Description ==

Chargely is the most powerful no code pricing table wordpress plugin in the market.

* The Chargely Subscription Plans WordPress plugin allows you to easily set up subscription plans for your website using the Chargely payment gateway. With this plugin, you can create and manage subscription plans, track user subscriptions, and accept recurring payments from your customers.

* Chargely provides you a dynamic set of tools that can help your business grow and reach unlimited potential. The Chargely Wordpress plugin is incredibly user friendly and provides pricing plan tables with an extensive set of features.

== Frequently Asked Questions ==

= Does Chargely support recurring payments? =

Yes, Chargely is a subscription service offering application that supports recurring payments.

= Does Chargely offer a free trial for new users? =

Yes, Chargely offers a 30-day free trial for new users.

= Does Chargely require credit card information to sign up for the free trial? =

No, You do not need to provide your credit card information while signing up for the free trial.

= Is it possible for the admin and customer to cancel their subscription at any point? =

Yes, there is an option for both admin and customer to cancel their subscription at any time based on selection by the merchant.

= Does this plugin require any coding knowledge to use? =

No, Chargely is a no code platform which eliminates the need for you to code.

== Installation ==

= Minimum Requirements =

* WordPress 5.6.10 or greater is recommended

1. Install Chargely either via the WordPress.org plugin directory or by uploading the files to your server.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After activation, you will see the plugin Chargely Subscriptions on the left sidebar of your WordPress dashboard.
4. Go to Chargely.com and create an account and Set up your payment gateway in Chargely.
5. Add the subscription plan shortcode to any page or post where you want to display the subscription plan.
That’s it. You’re ready to go!

Shortcodes
[chargely_subscription_plan id="123"] - Displays a subscription plan form for the plan with ID 123.
